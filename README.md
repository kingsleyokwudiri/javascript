# My JavaScript Learning Journey 🟨 🚀

Welcome to my repository! This is a personal archive of all the code, scripts, and practice exercises I created while teaching myself JavaScript.

## 📂 Project Structure

- `/Basics` - Variables, data types, if/else statements, and arrays.
- `/Functions_and_Objects` - Writing reusable code, scope, and JS objects.
- `/DOM_Manipulation` - Code that interacts with HTML/CSS to change web pages live.
- `/Mini_Projects` - Small interactive web apps and scripts.

## 🕹️ Projects I Built

- **🧮 Calculator**
  - A fully functional calculator that handles basic math operations.
  - _What I learned:_ Managing state, parsing strings, and updating the display screen.
- **🪙 Coin Flip Game**
  - A quick game of chance against the computer.
  - _What I learned:_ Using `Math.random()` to generate random choices and handling basic conditional logic.
- **🪨 Rock, Paper, Scissors**
  - The classic game played directly in the browser.
  - _What I learned:_ Comparing user input against computer choices, tracking game scores, and updating the game interface.

## 🏗️ Currently Working On...

  - _Current Goals:_ Mastering task management, working with arrays, and looking into saving data so tasks stay there even after refreshing the page (Local Storage).
